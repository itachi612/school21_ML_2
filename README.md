# Linear Regression Models

Проект посвящён изучению моделей линейной регрессии и регуляризации в задачах машинного обучения.

Основная цель проекта — разобраться, как работают линейные модели, как реализовать их самостоятельно, как сравнивать качество моделей и как бороться с переобучением с помощью регуляризации.

## Описание проекта

В рамках проекта рассматривается полный процесс решения задачи регрессии:

* загрузка и первичный анализ данных;
* подготовка признаков;
* реализация собственных моделей;
* сравнение собственных реализаций с моделями из `sklearn`;
* нормализация данных;
* регуляризация моделей;
* анализ переобучения и недообучения;
* сравнение качества моделей по метрикам.

## Основные темы

### Линейная регрессия

Линейная регрессия — это базовая модель машинного обучения, которая описывает зависимость целевой переменной от признаков с помощью линейной функции:

```text
y = wX + b
```

Модель подбирает такие веса, чтобы ошибка между реальными и предсказанными значениями была минимальной.

### Градиентный спуск

В проекте реализуется обучение модели с помощью градиентного спуска.
Этот метод позволяет постепенно обновлять веса модели в направлении уменьшения функции ошибки.

### Регуляризация

В проекте рассматриваются три регуляризованные модели:

* `Ridge Regression` — L2-регуляризация;
* `Lasso Regression` — L1-регуляризация;
* `ElasticNet` — комбинация L1 и L2-регуляризации.

Регуляризация помогает уменьшить переобучение и сделать модель более устойчивой.

### Нормализация признаков

Рассматриваются два способа нормализации данных:

* `MinMaxScaler`;
* `StandardScaler`.

Нормализация важна для линейных моделей, потому что признаки с разными масштабами могут по-разному влиять на обучение.

### Переобучение и недообучение

В проекте демонстрируется, как модель может переобучиться на сложных признаках, например на полиномиальных признаках высокой степени.

Также показывается, как регуляризация помогает снизить переобучение.

## Реализованные модели

В проекте реализованы собственные версии следующих моделей:

* Linear Regression;
* Ridge Regression;
* Lasso Regression;
* ElasticNet.

Также используются модели из `sklearn`:

* `LinearRegression`;
* `Ridge`;
* `Lasso`;
* `ElasticNet`.

## Метрики качества

Для оценки качества моделей используются следующие метрики:

* `MAE` — средняя абсолютная ошибка;
* `RMSE` — корень из среднеквадратичной ошибки;
* `R2 Score` — коэффициент детерминации.

Метрики считаются отдельно на train и test выборках, чтобы оценить качество обобщения модели.

## Установка и запуск

```bash
cd linear-regression-models
```

## Результаты

В проекте сравниваются собственные реализации моделей с реализациями из `sklearn`.

Если модели реализованы корректно, значения метрик должны быть близки к результатам библиотечных моделей.

Также показывается, что регуляризация помогает улучшить стабильность модели и снизить переобучение, особенно при использовании большого количества признаков или полиномиальных признаков.

## Основные выводы

* Линейная регрессия — простая и интерпретируемая базовая модель.
* Градиентный спуск позволяет обучать модель итерационно.
* Нормализация признаков важна для корректного обучения линейных моделей.

* L2-регуляризация уменьшает влияние слишком больших весов.
* ElasticNet объединяет свойства L1 и L2-регуляризации.
* Сравнение train и test метрик помогает выявить переобучение и недообучение.
